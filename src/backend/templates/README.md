# Component Templates

Use these copy-paste stubs when creating custom environment components:

- `action_processor_template.py`
- `observation_model_template.py`
- `reward_function_template.py`

## Quick start

1. Copy a template into your module (for example `plume_nav_sim/actions/my_action.py`).
2. Rename the `Example...` class.
3. Replace `TODO` blocks with your logic.
4. Keep method signatures unchanged so protocol checks still pass.

For full extension guidance and protocol references, see `../EXTENDING.md`.
